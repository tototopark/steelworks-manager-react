# 직원, 차량, 공휴일, 근태, 어드민 메뉴 비교 기록 보완 계획

본 계획은 레거시(1번 앱)와 신규 React(2번 앱) 간의 남은 메뉴 매핑 및 구현 격차를 상세히 분석하고 작업 결과를 기록하기 위한 계획입니다.

## 제안된 변경 사항

### 비교 분석서 보완

#### [MODIFY] [_수정_react_46_1번과2번앱메뉴비교표.md](file:///f:/pe/public_html/steelworks-manager-react/수정/_수정_react_46_1번과2번앱메뉴비교표.md)
- **Employees(직원)**: Active/Retired 탭 분리, 아바타 업로더, 임시 비밀번호 난수 발급 모달 연동 완료 상태 기록.
- **Vehicles(차량)**: WOF/REGO 30일 이내 만료 경고 알림 연동 및 km 마일리지 기반 스펙 관리 완료 상태 기록.
- **Holidays(공휴일)**: 백엔드 API 연동 상태 및 달력 바인딩 완료이나, 관리자 전용 신규 프론트엔드 UI 및 사이드바 메뉴가 누락된 격차 상태 명시 (추가 작업 필요).
- **Punch Clock(근태)**: Live NZ Clock 기반 원클릭 출퇴근(IN/OUT) 기록 기능 및 Timesheet 관리자 상세 필터 페이지 연동 완료 상태 기록.
- **Admin(어드민)**: Level 99 권한 가드, SQLite 테이블 익스플로러, 무결성 체크 및 데이터베이스 초기화/패스워드 초기화 기능 통합, Mermaid ERD 스키마 인터랙티브 그래프 이식 완료 상태 기록.
