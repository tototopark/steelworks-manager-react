    - 레거시 PHP 앱 `holidays.html` / `holidays.php`에 존재하는 공휴일 등록 및 삭제 기능이 React 앱에는 미구현 상태임.

  * **Job Sheet Edit & Excel Ingest (8.php / update_jobsdetails.php)**
    - 레거시 PHP 앱 `8.php`에서는 100라인의 그리드를 통해 Job Member(부재)들의 정보(Lot, Member명, Galv+Paint 여부, 공정별 견적 공수: Detailing, Fabrication, Install, Truck)를 직접 행별로 수정하거나 Excel raw 데이터를 Copy&Paste하여 대량으로 업데이트하는 기능이 존재함.
    - 현재 React 앱의 `Jobs` 상세화면은 단순한 Member 리스트 출력 및 가공 펀치(Start/Stop)만 제공하고 있어, 각 Member들의 Lot 번호, Member명, Galv/Paint, 4가지 공정 견적 시간(Detailing, Fabrication, Install, Truck)을 그리드 형태로 일괄 수정하고 저장하는 편집/대량 업로드 기능이 마이그레이션 누락된 상태임.
  * **Edit User / Register Profile (11.php / 12.php / register.php)**
    - 레거시 PHP 앱 `11.php` 및 `12.php`에서는 직원의 프로필(Avatar 파일, 로그인 ID, 비밀번호 Reset, 이름, Bay, right_level, shop_label, IP_1, IP_2, IP_3 접근 제어용 IP 리스트)을 일괄 그리드로 수정/저장할 수 있음.
    - 현재 React의 `Employees` 화면은 기본적인 직원 디렉토리 및 비밀번호 발급, 단일 이미지 업로더만 제공하므로, IP 리스트(ip_1, ip_2, ip_3) 관리 기능 및 일괄 편집 기능 보완이 필요함.
