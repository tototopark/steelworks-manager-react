## 2. 미마이그레이션 6대 요약 사항

1. **Holidays (공휴일 관리)**: 백엔드 API 작동 중이나 프론트 UI 페이지(holidays/page.js) 누락.
2. **Job Drawings/Photos (도면 및 사진)**: 백엔드 API 작동 중이나 Jobs 상세 보기 내 업로드/갤러리 컴포넌트 부재.
3. **직원 IP 접근 통제**: 직원 마스터의 IP 제한 필드 및 백엔드 미들웨어 검증 미이식.
4. **Job Sheet Edit (일괄 그리드 수정)**: 레거시 스타일의 100라인 단위 일괄 편집 그리드 포팅 필요.
5. **대시보드 시각화 및 페이징**: 대시보드 진행률 목록 50개 제한 및 오름차순 정렬, 진행 상태별 색상바 및 pcs 완료 비율 출력 보완 필요.
6. **비밀번호 강제 변경 UI**: 기본 패스워드 로그인 시 강제 변경 플로우 연결 검증 및 보완 필요.

본 작업을 통해 차기 세션의 원활한 마이그레이션 인수인계가 완료되었습니다.
